ALWAYS Verify Before Account Changes
Required for:
Password reset, MFA reset, unlock account, permission changes, any security action
Verification Process
1. Ask caller: "What is your employee ID number?"

2. Verify in Directory (Employees OU)

3. Ask: "What department do you work in?"

4. Cross-check AD department field

5. Ask: "Who is your direct manager?"

6. Verify manager name matches AD manager field

7. Send SMS verification code to phone number in AD

8. Caller must read back code verbatim

9. Only then: Proceed with account changes

Red Flags - Do NOT Proceed
Caller cannot answer basic questions

Phone number in AD doesn't match caller's number

Urgent/angry tone claiming emergency

Requesting access to another person's account

No manager information available in AD

Multiple failed verification attempts

Important Rules
Never skip verification even if caller pressures you

Never verify via email (too easy to spoof)

Always document verification in ticket notes

Refusal to verify = decline request and close ticket
